React Redux + Redux Toolkit Basic Explanation

React mein problem kya hoti hai?

Maan lo tumhare app mein user, cart, theme, login state waghera hain.

Agar yeh data har component mein alag alag bhejna pade, to code messy ho jata hai.

Yahin par Redux ka kaam aata hai.

Redux kya hota hai?

Redux ek central store hota hai jahan app ka common data rakha jata hai.

Socho:

  • React components = rooms
  • Redux store = ek central cupboard

Har component us cupboard se data le sakta hai aur update bhi kar sakta hai.

Redux ke 3 basic parts

1. Store

Jahan data rehta hai.

{
  count: 0
}

2. Action

Batata hai kya kaam karna hai.

{
  type: "increment"
}

3. Reducer

Actual mein state change karta hai.

if(action.type === "increment"){
   state.count += 1
}

Simple Flow

Component button click
→ action dispatch hoti hai
→ reducer run hota hai
→ store update hota hai
→ UI re-render hoti hai

Sabse basic example

Install

npm install @reduxjs/toolkit react-redux

Step 1 — Slice banana

features/counterSlice.js

import { createSlice } from "@reduxjs/toolkit";

const counterSlice = createSlice({
  name: "counter",
  initialState: {
    count: 0
  },
  reducers: {
    increment: (state) => {
      state.count += 1;
    },
    decrement: (state) => {
      state.count -= 1;
    }
  }
});

export const { increment, decrement } = counterSlice.actions;
export default counterSlice.reducer;

Yeh kya hua?

createSlice se mil gaya:

  • reducer
  • actions

Matlab alag alag files ki zarurat nahi.

Step 2 — Store banana

app/store.js

import { configureStore } from "@reduxjs/toolkit";
import counterReducer from "../features/counterSlice";

export const store = configureStore({
  reducer: {
    counter: counterReducer
  }
});

Step 3 — App ko provider dena

main.jsx

import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

import { Provider } from "react-redux";
import { store } from "./app/store";

ReactDOM.createRoot(document.getElementById("root")).render(
  <Provider store={store}>
    <App />
  </Provider>
);

Provider kya karta hai?

Yeh React ko bolta hai: “Redux store ab poori app mein available hai.”

Step 4 — Component mein use karna

App.jsx

import React from "react";
import { useSelector, useDispatch } from "react-redux";
import { increment, decrement } from "./features/counterSlice";

export default function App() {
  const count = useSelector((state) => state.counter.count);
  const dispatch = useDispatch();

  return (
    <div>
      <h1>{count}</h1>

      <button onClick={() => dispatch(increment())}>
        Increment
      </button>

      <button onClick={() => dispatch(decrement())}>
        Decrement
      </button>
    </div>
  );
}

Yeh 2 hooks bohat important hain

useSelector

Store se data nikalta hai.

const count = useSelector((state) => state.counter.count);

Matlab store ke andar:

{
  counter: {
    count: 0
  }
}

useDispatch

Action bhejta hai.

dispatch(increment())

Sabse important baat

Redux Toolkit mein normally yeh 4 cheezein hoti hain:

  • Slice → logic hota hai
  • Store → data ka ghar
  • Provider → app ko store deta hai
  • Hooks → useSelector aur useDispatch
